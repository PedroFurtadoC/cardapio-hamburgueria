Sobre a Royale Burger

A Royale Burger é uma hamburgueria fictícia desenvolvida como parte de um projeto acadêmico, para a matéria Desenvolvimento Web. Meu objetivo é demonstrar um sistema de cardápio digital moderno, funcional e responsivo para ambientes de restaurante.

Este projeto utiliza tecnologias como HTML, CSS, JavaScript, PHP, XAMPP e MySQL, integradas para sistema completo para facilitar a visualização de um cardápio bonito e interativo ao consumidor.

Embora fictícia, a Royale Burger reflete meu comprometimento em criar uma interface premium, com foco em funcionalidade e design moderno. Assim como a ideia da hamburgueria.
